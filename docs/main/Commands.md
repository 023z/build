# 常用命令
## 打印帮助信息
~~~bash
./bin/teaweb -h
~~~

## 打印版本信息
~~~bash
./bin/teaweb -v
~~~

## 启动服务
~~~bash
./bin/teaweb start
~~~

## 停止服务
~~~bash
./bin/teaweb stop
~~~

## 重启服务
~~~bash
./bin/teaweb restart
~~~

## 重新加载代理配置
v0.1.0以后支持
~~~bash
./bin/teaweb reload
~~~

## 重置服务状态
v0.0.8以后支持
~~~bash
./bin/teaweb reset
~~~

## 查看服务状态
v0.1.0以后支持
~~~bash
./bin/teaweb status
~~~
