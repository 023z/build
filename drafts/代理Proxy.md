# 代理
* 自动缓存静态资源：.xml、.css、.js、.zip、.map、fonts等等
* 自动从本机的nginx、vanish、squid、haproxy、apache、fpm、tomcat ...读取配置并添加到当前项目中

# 界面
~~~
|  已有服务    |  添加

|---------------|   |-------------|
|  web001       |   |  Name       |
|  nginx        |   |  Server     |   ...
|   [ok]        |   |  State      |
|  Chart        |   |  Chart      |
|---------------|   |-------------|

~~~
