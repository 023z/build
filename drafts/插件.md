# 插件系统
## 目标
提供一个简单可扩展的插件系统，让开发者只要会使用Go语言就能轻松开发属于自己的插件。

## 扩展点
* TopBar - 顶部菜单
* MenuBar - 左侧菜单
* HelperBar - 右侧菜单
* Dashboard - 控制台

## 实现
~~~
TeaWeb           <--->   Plugin1, Plugin2, ... 
LocalSocket
~~~
