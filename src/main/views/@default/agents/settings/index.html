部署Agent

请在要管理App的主机上下载并安装agent，并修改configs/agent.conf内容如下： 
master: http://TeaWeb访问地址
id: {{agent.id}}
key: {{agent.key}}
其中TeaWeb访问地址通常为当前TeaWeb所在服务器可访问的IP加上端口7777，比如当前服务器的IP是192.168.1.100，那么可以填上master: http://192.168.1.100:7777
